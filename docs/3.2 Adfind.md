Adfind用来查询域内资源，389端口

1、远程连接查询域控

```powershell
PS C:\Users\Administrator\Desktop\AdFind>
.\AdFind.exe -h 172.16.80.189 -u sec\administrator -up Az123456@ -sc dclist.

win2019-1.sec.com
PS C:\Users\Administrator\Desktop\AdFind>
```
